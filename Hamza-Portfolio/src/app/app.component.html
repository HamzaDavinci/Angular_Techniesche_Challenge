I have been working on a few projects since I was young, and you can see them here.

Hypelands

Hypelands is a project I started in 2017. It is a custom survival server, similar to Minecraft, but for the game Hytale. This game has not been released yet, but I have created a Discord community and a website. I worked with my own web server and connecting domains.

What did I learn from this project?

I learned how to maintain a community, how to use a web server, how DNS works, and how domains work.

#1 StarveLands

StarveLands #1 was a Minecraft server where you could survive against custom creatures in a world with different zones. There were 3 zones to explore and custom NPCs, monsters, and items to discover and craft. The goal of this game was to become the strongest by advancing to the stronger zones.

What did I learn from this project?

I learned how to maintain a community, how to make different plugins work together by modifying YML files, and I also learned how to create an online store for Minecraft servers.

#2 StarveLands

StarveLands #2 was not the same as the first StarveLands, but it was inspired by it, with NPCs, monsters, and items. This server featured a long map with 7 different zones where you had to complete quests to progress to the next zone and acquire new items. The goal was to defeat all the zones and eventually defeat a gorilla boss by becoming stronger in each zone.

What did I learn from this project?

I learned how to maintain a community, how to use a Minecraft server, how to link a domain with your Minecraft server IP, and how to work with YML files.
